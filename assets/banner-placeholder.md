# Banner Image Placeholder

Para adicionar um banner personalizado ao README:

1. Crie ou baixe uma imagem de banner (recomendado: 1200x400px)
2. Salve como `banner.png` nesta pasta
3. O README já está configurado para usar esta imagem

## Sugestões de Banner

O banner deve conter:
- Logo do Postman
- Logo do GitHub Copilot
- Logo do VS Code
- Texto: "Postman MCP + GitHub Copilot"
- Subtítulo: "Teste APIs com Linguagem Natural"

## Ferramentas para Criar Banner

- Canva (https://canva.com)
- Figma (https://figma.com)
- Photoshop
- GIMP (gratuito)

## Dimensões Recomendadas

- Largura: 1200px
- Altura: 400px ou 300px
- Formato: PNG ou JPG
- Fundo: Gradiente azul/laranja (cores do Postman e GitHub)
